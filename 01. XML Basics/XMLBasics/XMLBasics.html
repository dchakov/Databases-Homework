1. What does the XML language represents? What does it used for?

    • XML was designed to be both human- and machine-readable.
    • XML is a software- and hardware-independent tool for storing and transporting data.
    • XML document does not DO anything.XML is just information wrapped in tags.
    • XML is often used to separate data from presentation.

  4. Explore http://en.wikipedia.org/wiki/Uniform_Resource_Identifier to learn more about URI, URN and URL definitions.
    • A Uniform Resource Identifier (URI) is a string of characters which identifies an Internet Resource. The purpose of using an URI is to give the namespace a unique name.
    • The most common URI is the Uniform Resource Locator (URL) which identifies an Internet domain address.
    • Universal Resource Name (URN) is a URI that identifies a resource by name in a particular namespace.
